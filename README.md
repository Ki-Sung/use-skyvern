# use-skyvern
## Skyvern 이란? 
- Skyvern은 LLM(대규모 언어 모델)과 컴퓨터 비전을 사용하여 브라우저 기반 워크플로우를 자동화하는 오픈소스 프로젝트를 말함
- 한마디로 LLM과 비전을 사용하여 웹사이트의 정보를 추출 할 수 있는 툴
## Skyvern 작동 방식 
1. 사용자가 자연어로 입력(API에서도 동일하게 입력)
2. NLP 레이어가 명령을 분해하고, 에이전트 실행 계획을 생성
3. 에이전트가 브라우저를 실행해 화면을 캡쳐하고, 비전으로 입력 필드 및 버튼을 찾는 등 동작 구현 
4. 보고서 페이지로 이동해 원하는 데이터를 인식, 추출 
5. 추출 결과를 저장한 스키마(CSV/JSON 등)로 변환 → 실행한 로그와 AI 행동 요약도 같이 제공
## Skyvern 핵심 기능 
- 설명 가능한 AI: Skyvern의 AI 결정은 내장된 설명과 함께 제공되어 모든 작업에 대한 명확한 요악과 근거를 제공 
- 프록시 네트워크: Skyvern은 프록시를 지원하여 국가, 주 또는 우편 번호 수준의 타겟팅이 가능 → 특정 지역 데이터를 수집해야 하는 경우에 유용 
- 캡차(CAPTCHA) 지원: Skyvern은 복잡한 워크플로우를 완료하기 위해 CAPCHA를 지원 
- 인증: Skyvern은 2FA/TOTP를 포함한 사용자 계정 인증을 지원
- 데이터 추출: Skyvern은 CSV/JSON을 포함하여 사용자가 선택한 모든 스키마에 워크플로우 데이터를 추출 가능
